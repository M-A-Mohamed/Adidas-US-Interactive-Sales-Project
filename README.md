# ADIDAS-US-Interactive-Sales-Project Dashboard Description

This dashboard provides a comprehensive overview of the Adidas US sales performance, focusing on key metrics, product categories, regional sales, and sales methods per retailer.

## Key Performance Indicators (KPIs)

*   **Unit Sold:** 9648
*   **Total Operating Profit:** $332,134,761.45
*   **Total Sales:** $120,166,650
*   **Total of Operating Profit (aggregate):** $582,802,260.62

## Sales Breakdown by Product Category

The dashboard features a donut chart and a bar chart breaking down sales by gender and footwear/apparel type:

*   **Men's Street Footwear:** 23%
*   **Men's Athletic Footwear:** 17%
*   **Men's Apparel:** 14%
*   **Women's Street Footwear:** 14%
*   **Women's Athletic Footwear:** 12%
*   **Women's Apparel:** 20%

## Time-Series Analysis

A line chart titled "Total Sales per Q" tracks sales performance across quarters from 2020 to 2022, with specific quarterly operating profit values displayed for 2020 and 2021.

## Regional and Location Data

*   **Count of unit sold per region:** A bar chart shows unit sales across Midwest, Northeast, South, Southeast, and West regions.
*   **State/City Filters:** Interactive lists allow filtering data by specific states (e.g., Alabama, Alaska, Arizona, Arkansas, California, Colorado, Connecticut) and cities (e.g., Albany, Albuquerque, Anchorage, Atlanta, Baltimore, Billings, Birmingham).

## Sales Method per Retailer Analysis

A detailed bar chart analyzes sales methods (In-store, Online, Outlet) and total sales for various retailers, including:

*   West Gear: $15,672,650
*   Walmart: $5,504,850
*   Sports Direct: $2,956,625
*   Kohl's: $7,652,500
*   Foot Locker: $15,672,650
*   Amazon: $2,236,625

The chart also provides a percentage breakdown of sales methods for each retailer.
